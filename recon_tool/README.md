# Recon Tool (Python)

This is a simple reconnaissance tool for performing basic information gathering during the early phase of penetration testing or bug bounty.

# Features

- WHOIS Lookup
- DNS Enumeration (`A`, `NS`, and `MX` records using `dig`)
- IP Geolocation (via ipinfo.io)
- Reverse DNS Lookup
- Basic Nmap Fast Scan

# Usage

1. Clone the repository
2. Create and activate a virtual environment
